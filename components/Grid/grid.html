Grid

Grid provides the user with a system to layout elements efficiently on the screen in a orderly fashion.

Two-Column Grid

Two-column grid can be used by using 'two-items' in the grid container.

            <!-- Using class 'two-items' to use two-column grid. -->

            <div class="two-items">
                <div class="grid-items item-1">Item 1</div>
                <div class="grid-items item-2">Item 2</div>
                <div class="grid-items item-3">Item 3</div>
                <div class="grid-items item-4">Item 4</div>
                <div class="grid-items item-5">Item 5</div>
                <div class="grid-items item-6">Item 6</div>
            </div>
