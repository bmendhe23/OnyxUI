Card

Cards are one of those components which are widely used to show instant summary of the objects in the form of images, title, description, etc.

They should be informative enough to give a clear idea about the object it is trying to present to the user.

Text only Card

This card only contains the title and description of the object. Text only cards can be used by using class 'text-card' in the card container.

Text Card Title
This is the card description which contains a brief summary of the title mentioned above. It give the user a idea about the title. 
                        
            <!-- Using class 'text-card' followed by various card elements. -->

            <div class="text-card">
                <div class="card-title">
                    Text Card Title
                </div>

                <div class="card-description">
                    This is the card description which contains a brief summary of the 
                    title mentioned above. It give the user a idea about the title.
                </div>
            </div>

Simple Card

Simple card contains a image, title, description and action buttons to it. This can be used by adding class 'card' to the container it is in.

Anime taking over the world
By Bhushan Mendhe
Ten reasons why anime will soon take over the animation industry and why anime has started becoming the most watched animated shows all over the world. 
                        
            <!-- Using class 'card' followed by various card elements to use the card-->

            <div class="card">
                <div class="card-img">
                    <img src="/assets/card/anime.jpg" alt="anime-card">
                </div>

                <div class="card-info">
                    <div class="card-title">
                        Anime taking over the world
                    </div>

                    <div class="card-author">
                        By Bhushan Mendhe
                    </div>

                    <div class="card-description">
                        Ten reasons why anime will soon take over the animation 
                        industry and why anime has started becoming the most watched 
                        animated shows all over the world.
                    </div>

                    <div class="card-actions">
                        <button class="btn btn-primary">
                            READ MORE
                        </button>

                        <button class="btn btn-secondary">
                            BOOKMARK
                        </button>
                    </div>
                </div>  
            </div>

Dismissible Card

Dismissible Cards allows user the option to dismiss the card when they don't want to see it. Dismissible Cards can be used by adding the class 'dismiss-card' to the card container after the card variant class.

Text Card Title
This is the card description which contains a brief summary of the title mentioned above. It give the user a idea about the title. 
                        
            <!-- Adding class 'dismiss-card' to the card variant and 'btn-dismiss' to the dismiss button. -->

            <div class="text-card dismiss-card">

                <button class="btn-dismiss">
                    <span class="material-icons">close</span>
                </button>

                <div class="card-title">
                    Text Card Title
                </div>

                <div class="card-description">
                    This is the card description which contains a brief summary of the 
                    title mentioned above. It give the user a idea about the title.
                </div>

            </div>

Text Overlay Card

Text overlay cards contain a text-overlay over a image. It is more like the simple card but some text is over the image. Text overlay card can be used by using class 'overlay-card' in the card container.

Anime taking over the world
By Bhushan Mendhe
Ten reasons why anime will soon take over the animation industry and why anime has started becoming the most watched animated shows all over the world.

                </div>

                    </div>

                    </div>
