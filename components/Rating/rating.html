Rating

Ratings are used to rate the product or a organization.

Simple Rating

Simple rating can be added by using class 'simple-rating' followed by variant class.

star 4.5
star 3.0
star 3.0 
                        
            
            <!-- Using class 'simple-rating' followed by variant class to use rating. -->

            <div class="simple-rating primary">
                <i class="material-icons">star</i>
                4.5
            </div>

            <div class="simple-rating danger">
                <i class="material-icons">star</i>
                3.0
            </div>

            <div class="simple-rating default">
                <i class="material-icons">star</i>
                3.0
            </div>

Star Rating

Star rating can be used by using the class 'star-rating' with class 'filled' for stars given by user.

star star star star star 
                        
            <!-- Using class 'star-rating' and 'filled' to use star rating. -->

            <div class="star-rating">
                <i class="material-icons filled">star</i>
                <i class="material-icons filled">star</i>
                <i class="material-icons filled">star</i>
                <i class="material-icons filled">star</i>
                <i class="material-icons">star</i>
            </div>
