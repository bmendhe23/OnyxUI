Lists

Lists help to categorize information points. They help to deduce the information to the user.

Spaced List

Spaced list can be used by adding class 'spaced-list' to the list container.

  1. List Item 1
  2. List Item 2
  3. List Item 3
  4. List Item 4
                        
            <!-- Using class 'spaced-list' in list container to use spaced list. -->

            <ol class="spaced-list">
                <li>List Item 1</li>
                <li>List Item 2</li>
                <li>List Item 3</li>
                <li>List Item 4</li>
            </ol>

Stacked List

Stacked list are lists that are stacked on each other. These are used in notifications. Stacked list can be used by using class 'stacked-list' in the list container with class 'link-item'

  • Bhushan Mendhe started following you! Mark as Read close
  • Rahul Singh started following you! Mark as Read close
  • Ankur Bohra started following you! Mark as Read close
  • Nilanjana Thakur started following you! Mark as Read close 
                        
            <!-- Using class 'stacked-list' and 'list-item' respectively as shown together to make a stacked list. -->
