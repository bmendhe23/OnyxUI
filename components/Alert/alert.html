Alert

Alert is used to redirect user's attention to an important message without disturbing the normal flow of the webpage and user.

Simple Alert

Simple alerts can be used my adding class 'alert' followed by the variant name.

This is a Primary Alert!
This is a Success Alert!
This is a Error Alert!
This is a Warning Alert! 
                        
            <!-- Using class 'alert' followed by variant name to use the Alert component-->

             
            <div class="alert primary">
            This is a Primary Alert!
                <button class="alert-action-btn">
                    <i class="simple material-icons">close </i>
                </button>
            </div>

            <div class="alert success">
            This is a Success Alert!
                <button class="alert-action-btn">
                    <i class="simple material-icons">close </i>
                </button>
            </div>

            <div class="alert error">
            This is a Error Alert!
                <button class="alert-action-btn">
                    <i class="simple material-icons">close </i>
                </button>
            </div>

            <div class="alert warning">
            This is a Warning Alert!
                <button class="alert-action-btn">
                    <i class="simple material-icons">close </i>
                </button>
            </div>
