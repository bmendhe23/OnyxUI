Avatar

Avatars are graphical representation of the user's profile picture usually rounded in shape. It can contain visual identity of users, projects, spaces, etc.

Basic Avatar

These basic avatars can be created by adding the class 'avatar' to the img tag for which you want to make a avatar.

                        
            <!-- Using class 'avatar' to use the basic avatar-->

            <img class="avatar" src="/assets/avatar/sparrow.jpg" alt="avatar-1">
            <img class="avatar" src="/assets/avatar/person.jpg" alt="avatar-2">
            <img class="avatar" src="/assets/avatar/sea.jpg" alt="avatar-3">

Varied Size Avatar

These avatars can be used apart from the default option by adding the class 'avatar-{sm|m|l}' to the img tag for which you want to make a avatar.

                        
            <!-- Using class 'avatar-{ sm | md | lg }' to use the varied size avatar-->

            <img class="avatar-sm" src="/assets/avatar/sparrow.jpg" alt="avatar-1">
            <img class="avatar-md" src="/assets/avatar/person.jpg" alt="avatar-2">
            <img class="avatar-lg" src="/assets/avatar/sea.jpg" alt="avatar-3">

Square Avatar

Avatars can be made square shaped by adding the class 'square' to the already existing class 'avatar-{sm|md|lg}' in the img tag for which you want to make a avatar.

                        
            <!-- Add class 'square' to use the square shaped avatars-->

            <img class="avatar-sm square" src="/assets/avatar/sparrow.jpg" alt="avatar-1">
            <img class="avatar-m square" src="/assets/avatar/person.jpg" alt="avatar-2">
            <img class="avatar-l square" src="/assets/avatar/sea.jpg" alt="avatar-3">

Initials Avatar

Avatars with just initials can be added using the class 'avatar-init' to the img tag for which you want to make a avatar.

BM
SM
RS
TP
PP
                        
            <!-- Add class 'square' to use the square shaped avatars-->

            <div class="avatar-init">BM</div>
            <div class="avatar-init">SM</div>
            <div class="avatar-init">RS</div>
            <div class="avatar-init">TP</div>
            <div class="avatar-init">PP</div>
