Button

Buttons are very important part of a website which enhance the interactivity of the user with various components of the page.

Classic Button

These buttons are primary to your app use and one of the most used in the app. Classic buttons can be used by adding class 'btn' for default or followed by variant class name.

                        
            <!-- Using class 'btn' followed by variant name to use the various classic buttons-->

            <button class="btn">DEFAULT</button>
            <button class="btn btn-primary">PRIMARY</button>
            <button class="btn btn-secondary">SECONDARY</button>
            <button class="btn btn-success">SUCCESS</button>
            <button class="btn btn-error">ERROR</button>
            <button class="btn btn-warning">WARNING</button>

Light Classic Button

This is the light colored version of the Classic Button. Add class 'light' to the variant class name after 'btn' class.

                        
            <!-- Adding class 'light' to the variant class after btn class.-->

            <div class="code-visual-container">
                <button class="btn btn-light">DEFAULT</button>
                <button class="btn btn-primary-light">PRIMARY</button>
                <button class="btn btn-success-light">SUCCESS</button>
                <button class="btn btn-error-light">ERROR</button>
                <button class="btn btn-warning-light">WARNING</button>
            </div>

Outline Button

Outline Buttons can be used by adding class 'outline' to the variant class name after 'btn' class.

                        
            <!-- Adding class 'outline' to the variant class after btn class.-->

            <button class="btn btn-outline">DEFAULT</button>
            <button class="btn btn-primary-outline">PRIMARY</button>
            <button class="btn btn-secondary-outline">SECONDARY</button>
            <button class="btn btn-success-outline">SUCCESS</button>
            <button class="btn btn-error-outline">ERROR</button>
            <button class="btn btn-warning-outline">WARNING</button>

Link Button

Link Buttons can be used by adding class 'btn-link' after 'btn' class.

                        
            <!-- Adding class 'btn-link' after btn class.-->

            <button class="btn btn-link">LINK</button>
            <button class="btn btn-link-light">LINK</button>
            <button class="btn btn-link-outline">LINK</button>

Icon Button

Icon Buttons can be used by adding class 'btn-icon' after 'btn' class.

                        
            <!-- Adding class 'btn-icon' after btn class.-->

            <button class="btn btn-icon">
                <i class="material-icons">star_rate</i>
                STAR
            </button>

            <button class="btn btn-primary btn-icon">
                <i class="material-icons">info</i>
                INFO
            </button>

            <button class="btn btn-success btn-icon">
                <i class="material-icons">lock</i>
            </button>

            <button class="btn btn-success btn-icon">
                <i class="material-icons">verified</i>
            </button>

            <button class="btn btn-error btn-icon">
                DELETE
                <i class="material-icons">delete</i>
            </button>

            <button class="btn btn-warning btn-icon">
                WARNING
                <i class="material-icons">warning</i>
            </button>

Float Button

Float Buttons can be used by adding class 'btn-float' after 'btn' class.

Note: Position of the float button can be set by modifying the - top, right, bottom, and left properties for btn-float class.
Here it has been positioned according to the float container.

                        
            <!-- Adding class 'btn-float' after btn class.-->

            <div class="float-container">
                <button class="btn btn-primary btn-icon btn-float">
                    <i class="material-icons">arrow_upward</i>
                </button>

                <button class="btn btn-error btn-icon btn-float">
                    <i class="material-icons">add</i>
                </button>

                <button class="btn btn-success btn-icon btn-float">
                    <i class="material-icons">group_add</i>
                </button>
            </div>
