Navigation

Navigation components are a crucial part of a web app that let you navigate to various parts of the web app.

Simple Navigation

Simple navigation can be used by using class 'navbar' in the navigation container.

Onyx UI
Home About us Contact us 
                        
            <!-- Using class 'navbar' to use navigation. -->

            <nav class="navbar">
                <section class="brand">
                    <a href="#">
                        <img src="logo.png" alt="OnyxUI" class="brand-logo">
                    </a>
                    <span class="brand-name"><a href="#">Onyx UI</a></span>
                </section>

                <section class="nav-links">
                    <a href="#">Home</a>
                    <a href="#">About us</a>
                    <a href="#">Contact us</a>
                </section>
            </nav>
