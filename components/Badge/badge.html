Badge

Badges are additional components added to another component that show information related to the component they are added to. They can be added to show status, count, etc.

Status Badge

The status can be added to the user avatars by adding the class 'badge' to the container which contains the avatar and adding the variant accordingly as show.

Note: This has 5 variants - online | offline | dnd | idle | invisible

                        
            <!-- Using class 'badge' and variant class to use the status badge-->

            <div class="badge">
                <img class="avatar" src="/assets/avatar/sparrow.jpg" alt="avatar-1">
                <div class="online"></div>
            </div>

            <div class="badge">
                <img class="avatar" src="/assets/avatar/person.jpg" alt="avatar-2">
                <div class="offline"></div>
            </div>

            <div class="badge">
                <img class="avatar" src="/assets/avatar/sea.jpg" alt="avatar-3">
                <div class="dnd"></div>
            </div>

            <div class="badge">
                <img class="avatar" src="/assets/avatar/old man.jpg" alt="avatar-4">
                <div class="idle"></div>
            </div>

            <div class="badge">
                <img class="avatar" src="/assets/avatar/river.jpg" alt="avatar-5">
                <div class="invisible"></div>
            </div>

Text Badge

These badges can be put over a text by adding the class 'text' to the div in the badge.

Note: This has two variants - Default: text | text-sm | text-lg

This is a text with small text Badge
5
This is a text with default text Badge
5
This is a text with large large text Badge
5
                        
            <!-- Using class 'badge' to use variant class of text badge to use text badge-->

            <div class="badge">
                <div>This is a text with small text Badge</div>
                <div class="text-sm">5</div>
            </div>

            <div class="badge">
                <div>This is a text with default text Badge</div>
                <div class="text">5</div>
            </div>

            <div class="badge">
                <div>This is a text with large large text Badge</div>
                <div class="text-lg">5</div>
            </div>

Number Badge

These can be used to show the no. of items present in the parent element. This badge can be added using the class 'number' with the parent element having badge class. This can be used on icons, avatars, etc.

Note: This has a variant - Add class 'lt' for number badge on left side.
