Image

Images give the user visual information related to any topic. They in a way also enhance the user experience as the web pages also look good having images in them.

Round Image

Round images can be used by using class 'round-img' to the image which you want to make round.

                        
            <!-- Using class 'round-img' to use the round image-->

            <img class="round-img" src="img-url" alt="round-img-1">
            <img class="round-img" src="img-url" alt="round-img-2">

Square Image

Square images can be used by using class 'square-img' to the image which you want to make square.

                        
            <!-- Using class 'square-img' to use the square image. -->

            <img class="square-img" src="img-url" alt="square-img-1">
            <img class="square-img" src="img-url" alt="sqaure-img-2">

Responsive Image

Images can be made responsive by using class 'responsive-img' to the image which you want to make responsive.

                        
            <!-- Using class 'responsive-img' to make the image responsive. -->

            <img class="responsive-img" src="img-url" alt="responsive-img">
