Snackbar

Snackbars provide a brief message about the task that is being carrying out.

Baseline Snackbar

Baseline snackbar displays brief message related to the running task on the bottom of the page. Baseline Snackbar can be used by using class 'baseline-snackbar' in the snackbar container.

Can't send photo. Retry in 5 seconds. close 
                        

            <!-- Using class 'baseline-snackbar' to use baseline snackbar. -->

            <div class="baseline-snackbar">
                Can't send photo. Retry in 5 seconds.
                <button>RETRY</button>
                <i class="snackbar-close material-icons">close</i>
            </div>

Top Snackbar

Top snackbar displays brief message related to the running task on the top of the page. Top Snackbar can be used by using class 'top-snackbar' in the snackbar container.

File was uploaded successfully. close 
                        
            <!-- Using class 'top-snackbar' to use top snackbar. -->

            <div class="top-snackbar">
                File was uploaded successfully.
                <i class="top-snackbar-close material-icons">close</i>
            </div>

Leading Snackbar

Leading Snackbar can be used by using class 'leading-snackbar' in the snackbar container.

Your photo has been archieved. close 
                        
            <!-- Using class 'leading-snackbar' to use top snackbar. -->

            <div class="leading-snackbar">
                Your photo has been archieved.
                <i class="leading-snackbar-close material-icons">close</i>
            </div>
