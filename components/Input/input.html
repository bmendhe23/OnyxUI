Input

Input fields allow the user to input data in order to get something when used in a form. They are in general used to get user response.

Normal Input

Normal input fields can be use used by using class 'input' to the input field.

                        
            <!-- Using class 'input' to use the normal input. -->

            <div class="input-container">
                <input class="input" type="text" placeholder="Username">
                <input class="input" type="text" placeholder="Password">
            </div>

Input with Icon

Input fields with icon can be use used by using class 'input-icon' to the input field.

email
lock 
                        
            <!-- Using class 'input-icon' to use the input field with icon. -->

            <div class="input-container">
                <div class="input-icon">
                    <i class="material-icons">email</i>
                    <input  class="input" type="text" placeholder="Username">
                </div>
                <div class="input-icon">
                    <i class="material-icons">lock</i>
                    <input  class="input" type="text" placeholder="Password">
                </div>
            </div>

Input with Validation Style

Input with validation style can be used by using the class 'valid-input' to the container containing the validation message.

Valid Email check 
                        
            <!-- Using class 'valid-input' to the container containing validation msg. -->

            <div class="input-container">
                <div class="valid-input">
                    <input class="input" type="email" placeholder="E-mail">
                    <div class="validation-msg">
                        Valid Email
                        <i class="material-icons">check</i>
                    </div>
                </div>
            </div>

Input with Error Style

Input with error style can be used by using the class 'error-input' to the container containing the error message.

Invalid Password close 
                        
            <!-- Using class 'error-input' to the container containing the error message. -->

            <div class="input-container">
                <div class="error-input">
                    <input class="input" type="password" placeholder="*******">
                    <div class="error-msg">
                        Invalid Password
                        <i class="material-icons">close</i>
                    </div>
                </div>
            </div>

Round Input

Round input fields can be use used by using class 'round-input' to the input field.

                        
            <!-- Using class 'round-input' to use the round input. -->

            <div class="input-container">
                <input class="round-input" type="text" placeholder="Username">
                <input class="round-input" type="text" placeholder="Password">
            </div>

Checkbox

Checkbox allows the user to select one or more options out of the available options. Checkbox can be use used by using class 'checkbox' to the container containing checkbox.

                        
            <!-- Using class 'checkbox' to the container containing checkbox. -->

            <div class="checkbox">
                <label for="option-1">
                    <input type="checkbox">
                    Option 1
                </label>
            </div>

            <div class="checkbox">
                <label for="option-2">
                    <input type="checkbox">
                    Option 2
                </label>
            </div>

            <div class="checkbox">
                <label for="option-3">
                    <input type="checkbox">
                    Option 3
                </label>
            </div>

Radio

Radio allows the user to select one option out of the available options. Radio can be use used by using class 'radio' to the container containing radio.

                        
            <!-- Using class 'radio' to the container containing radio. -->

            <div class="radio">
                <label for="option-1">
                    <input name="option" type="radio">
                    Option 1
